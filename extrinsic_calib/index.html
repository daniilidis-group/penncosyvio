Extrinsic Calibration

All sensors are extrinsically calibrated with respect to the rig body (B) reference frame (which coincides with the center GoPro C2 camera). For each sensor we give the transform from the rig coordinate system to the sensor coordinate system. A left superscript indicates the coordinate system in which vectors are expressed. For example, \(^{\mathrm{C1}}\mathbf{X}\) are the coordinates of vector X in the reference frame of camera C1. The transform S2 TS1 takes a vector expressed in coordinate system S1 and transforms it to system S2: S2X = S2TS1 S1X. For example, given coordinates in the rig reference frame, the ones in sensor system S1 can be obtained via: \[ ^{\mathrm{S1}}\mathbf{X} = ^{\mathrm{S1}}\mathbf{T}_{\mathrm{B}}\ ^{\mathrm{B}}\mathbf{X} \]

S1X = S1TB BX

The transform T can be expressed as a 3x4 transformation matrix:

T = [R, t]

where t is a 3x1 translation vector, and R is a pure rotation.
